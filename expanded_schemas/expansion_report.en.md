# Schema Expansion Report

中文源文件：[expansion_report.md](expansion_report.md)

This is the English companion for the generated schema expansion report.

## Scope

The expansion report records how source schemas under `schemas/` are expanded into derived schema artifacts.

## Regeneration

From the repository root:

```bash
make expand
```

## Maintenance

- Treat generated schema reports as derived artifacts.
- Fix source schema or generator issues rather than editing expanded outputs manually.
- Keep this English companion aligned when the report scope changes.
