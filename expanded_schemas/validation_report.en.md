This is the English companion for the generated expanded schema validation report.

## Scope

The validation report records validation results for expanded UModel schema artifacts.

## Regeneration

From the repository root:

```bash
make expand
```

## Maintenance

- Treat expanded schema reports as generated artifacts.
- Keep schema source, generated output, validation reports, and documentation aligned.
- Update this English companion if the validation scope changes.
