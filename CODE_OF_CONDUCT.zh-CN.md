# 行为准则

English version: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

UModel 是一个开源技术社区。我们希望讨论、issue、pull request 和 review 都保持有用、尊重，并以证据为基础。

## 期望行为

- 保持尊重和建设性。
- 默认善意，但当行为造成伤害或混乱时及时澄清。
- 技术分歧聚焦事实、取舍和项目目标。
- 给维护者和贡献者足够上下文复现问题。
- 尊重并标注想法、报告和贡献来源。

## 不可接受行为

- 骚扰、威胁、人身攻击或歧视性语言。
- 未经允许公开私人信息。
- 反复打断或偏离技术讨论。
- 垃圾信息、商业滥用或恶意协同行为。
- 在维护者完成 triage 前公开披露安全漏洞。

## 执行

维护者可以 moderation、编辑、隐藏或移除违反行为准则的内容。严重或重复违规可能导致临时或永久失去参与权限。

## 报告

行为问题请通过托管组织的维护者私有联系方式报告。安全问题应遵循 [SECURITY.zh-CN.md](SECURITY.zh-CN.md)。
